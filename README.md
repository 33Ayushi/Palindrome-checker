# Palindrome-checker
Super simple, free and fast browser-based utility for checking if text is a palindrome. Just paste your text and you'll instantly get its status.
Palindrome Checker 🔁

A simple and fast browser-based utility that checks whether a given text is a palindrome or not.
Just enter or paste your text, click the check button, and instantly see the result.

A palindrome is a word, phrase, or number that reads the same forward and backward (for example: madam, racecar).

✨ Features

Instantly check if text is a palindrome

Clean and simple user interface

Works directly in the browser

Fast and lightweight
